# FairHouseMap
Website about a redistricted and more fair map of the 2022 US House of Representatives.


Map data in /data/ as a topojson file.


Libraries used:
- d3.js
- topojson


Map made with [Dave's Redistricting](https://davesredistricting.org/maps#home).
